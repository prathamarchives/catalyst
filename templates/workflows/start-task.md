# start-task

## job

Classify task, select load mode, identify needed brain files, and name success criteria before working.

## inputs

Catalyst Brain files, task request, output draft, user feedback, improvement-log.

## steps

1. identify relevant brain files
2. apply written rules explicitly
3. produce or update the artifact
4. check against standards
5. log what changed and why

## output

A completed task or a precise brain/update patch that improves the next task.
