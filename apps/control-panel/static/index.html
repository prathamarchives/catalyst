Step 4 · Permission

Catalyst found possible sources.

Discovery is read-only — it checks where things live and counts categories. It never reads file contents. You decide the scan.

Safe recommended scan reads AI sessions, notes, and markdown workspaces. It excludes secrets, tokens, private DMs, client data, binaries, vendor/build folders, and dependency folders.
Content scanning is wired but not auto-run in v0.3 — your approval is recorded for the agent step.

Step 8 · Agents

Let many agents use the brain.

Catalyst ships a local MCP scaffold so agents can list and read sections, review output, append feedback, and propose updates — without giving them raw file access.

Scaffold, honestly. The MCP server (tools/mcp_server.py) speaks JSON-RPC over stdio and implements the five tools below. It is local-only, no network, and writes only through feedback/proposal functions. It is a working scaffold, not a certified MCP build — see docs/mcp.md.
py tools/mcp_server.py        # stdio MCP scaffold (local-only)
