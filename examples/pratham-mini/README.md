# pratham-mini — worked example

A small, real Creative Brain built for Pratham — the builder of this repo and its user zero. Trimmed to public-safe material, kept short on purpose: this shows the *shape* of a finished run, not the full depth of a real one.

What's here:

- `creative-brain/` — all ten files, filled with real (public) extracted material; `feedback-memory.md` shows the four-section lifecycle (active rules / raw log / retired / distillation log)
- `skills/creative-identity-skill.md` — the generated skill, customized to this user, with quick + full load modes
- `workflows/` — all five workflows as they land in an output folder (use / update-from-feedback / review / blind-ab-test / distill-memory)
- `evals/blind-ab-log.md` — the blind preference scoreboard with the running brain_win_rate
- `before-after.md` — the proof artifact: same task, competent generic baseline vs brain-loaded, with real feedback and the memory update it produced

Read `before-after.md` first. It's the point of the whole system: same model, different brain, visible difference.
