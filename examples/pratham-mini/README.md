# pratham-mini — worked example

A small, real Creative Brain built for Pratham — the builder of this repo and its user zero. Trimmed to public-safe material, kept short on purpose: this shows the *shape* of a finished run, not the full depth of a real one.

What's here:

- `creative-brain/` — all ten files, filled with real (public) extracted material
- `skills/creative-identity-skill.md` — the generated skill, customized to this user
- `workflows/` — the three workflows as they land in an output folder
- `before-after.md` — the proof artifact: same task, generic vs brain-loaded, with real feedback and the memory update it produced

Read `before-after.md` first. It's the point of the whole system: same model, different brain, visible difference.
