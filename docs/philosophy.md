# philosophy

Models are shared infrastructure. The differentiator is the operating context around them: identity, standards, judgment, constraints, taste, and feedback.

Catalyst exists because agents reset too easily. They can follow a prompt for one task, but unless corrections become durable rules and checks, the next session forgets the standard again.

A Catalyst Brain gives the agent a local standard to work from and a loop for improving it:

```txt
task → load brain → produce → review against standards → user feedback → update brain/skill/eval → sharper next task
```

The point is not style imitation. Voice matters, but only as one signal inside a larger system: who the agent represents, what work is good enough, when to push back, and what previous corrections taught it.
