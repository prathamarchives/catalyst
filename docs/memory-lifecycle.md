# memory lifecycle

Catalyst memory is maintained, not merely appended.

## Active rules

Current, high-confidence rules loaded often.

## Raw feedback log

Raw user quotes and task context. Preserve evidence.

## Retired rules

Stale or superseded context. Retire with dates and reasons.

## Distillation log

Record when duplicates were merged, repeated corrections promoted, contradictions surfaced, and stale context decayed.

Memory distillation must merge duplicate rules, handle stale context, preserve raw evidence, and trigger contradiction review instead of silently guessing.
