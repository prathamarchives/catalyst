# Catalyst Brain

A Catalyst Brain is a local, agent-readable operating layer for identity, context, standards, judgment, and adaptive feedback.

It is not a writing style profile. It is the system an agent uses to answer:

- who/what am I representing?
- what does the user want here?
- what standard must this meet?
- what should I reject or push back on?
- what did the user correct last time?
- what should update so this mistake does not repeat?

The most valuable addition is **task-time evaluation**: every meaningful output is checked against user-specific standards before and after the user reacts. That turns feedback into compounding agent behavior.
