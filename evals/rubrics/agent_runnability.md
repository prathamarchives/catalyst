# agent runnability rubric

Score whether a fresh agent can run the repo:

1. first action is discovery
2. reading contents is permission-gated
3. output structure is explicit
4. templates are protected
5. skills define quick mode and full mode
6. task-time evaluation is required for real tasks
7. feedback update loop is explicit
8. privacy caveats are visible
9. eval command is documented
10. no app/dashboard/API is implied for v0
