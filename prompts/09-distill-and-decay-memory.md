# 09 — distill and decay memory

Append-only memory rots. Run this every 10 feedback entries or whenever feedback-memory becomes noisy.

## Process

- merge duplicate rules
- promote repeated corrections into standing standards
- retire stale context with dates
- preserve durable taste and judgment
- surface contradictions for user review
- keep raw quotes attached to distilled rules
- split feedback-memory if needed into active rules, raw feedback log, retired rules, and distillation log

Never silently resolve contradictions. Ask the user or mark open-question.
