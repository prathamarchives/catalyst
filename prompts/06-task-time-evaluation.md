# 06 — task-time evaluation

Use this on real tasks. Do not stage fake comparisons.

## Loop

1. Classify the task.
2. Load relevant Catalyst Brain files.
3. Produce the work.
4. Run output-review:
   - identity alignment
   - standards fit
   - judgment check
   - constraint check
   - task-pattern fit
   - anti-slop scan
5. Revise if the output fails the written standard.
6. Show the user.
7. Capture explicit and implicit feedback.
8. Update Catalyst Brain, skills, evals, and improvement-log.

A task is not complete until the review is run or an honest blocker is named.
